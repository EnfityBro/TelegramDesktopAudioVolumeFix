# TelegramDesktopAudioVolumeFix

> [!NOTE]
> Automatic volume increase of the system during audio playback in Telegram Desktop.


## Requirements
- Windows 10+


## How to use
1. Download actual zip archive from [releases](https://github.com/EnfityBro/TelegramDesktopAudioVolumeFix/releases).
2. Unzip to any convenient folder on your computer.
3. Launch 'TelegramDesktopAudioVolumeFix.exe' (or put it in autorun).
